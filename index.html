Dexy's R.E.P.O Modpack Installation

1

Installiere R2ModMan

download R2ModMan v3.1.57
2

Suche nach "Repo" im Game Selection Screen

Klicke auf "Set as default"

3

Öffne den Profile Selection Screen

Klicke auf "Import/Update"

4

Wähle "From code"

5

Gib den Code ein

01961743-5e6d-d6df-f926-bc6779363ac6

Klicke auf "Continue"

6

Klicke auf "Import"

7

Wähle den Profil-Typ

Klicke auf "Import new profile" oder "Update existing profile"

8

Klicke auf "Create"

9

Wähle das neue Profil

Klicke auf "Select Profile"

10

Starte das Modpack

Klicke auf "Start modded"

Bereit zum Spielen!

Du hast deine Mods erfolgreich synchronisiert! Du kannst jetzt modifiziert mit deinen Freunden spielen!

info Hinweis: Stelle sicher, dass das Basisspiel installiert ist, bevor du die modifizierte Version startest.
